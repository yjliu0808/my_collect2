1 练习使用内连接查询

1.1 问题

查询emp表，dept表及salgrade表的信息，完成下列SQL语句的编写：

1）查询emp表中员工的编号，姓名，职位，及对应部门的编号，名称；

2）查询出员工的名称、部门名称；

3）列出员工的名字、薪水、薪水的级别；

4）查询出员工的名字和领导名字。

1.2 方案

关联查询：

所需要查询的数据来源多张表，需要根据某种规则，把这些表中的数据进行连接，完成的数据。

内连接：

select 别名1.列名/* , 别名2.列名/*

from 表名1 别名1，表名2 别名2

where 关联条件；

内连接-等值连接：

在where中关联关系用“=”

内连接-非等值连接：

两张表的关系不是等值关系，除了=之外，比如 > < >= <=

from 表1 别名1，表2 别名2， …

where 非等值连接条件；

内连接-自连接：

如果一张表中两个字段之间有关系，可以采用自连接

1.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：查询emp表中员工的编号，姓名，职位，及对应部门的编号，名称

查询emp表中员工的编号，姓名，职位，及对应部门的编号，名称，SQL语句如下所示：

copytextpop-up
  1. select e.empno,e.ename,e.job,d.deptno,d.dname,d.loc
  2. from emp e,dept d
  3. where e.deptno = d.deptno;
select e.empno,e.ename,e.job,d.deptno,d.dname,d.loc
from emp e,dept d
where e.deptno = d.deptno;

步骤二：查询出员工的名称、部门名称

查询出员工的名称、部门名称，SQL语句如下所示：

  2. from emp e, dept d
  3. where e.deptno = d.deptno; 
select e.ename, d.dname
from emp e, dept d
where e.deptno = d.deptno;

步骤三：列出员工的名字、薪水、薪水的级别

列出员工的名字、薪水、薪水的级别，SQL语句如下所示：

copytextpop-up
  1. select e.ename, e.sal, g.grade
  2. from emp e, salgrade g
  3. where e.sal between g.losal and g.hisal;
select e.ename, e.sal, g.grade
from emp e, salgrade g
where e.sal between g.losal and g.hisal;

步骤四：查询出员工的名字和领导名字

查询出员工的名字和领导名字，SQL语句如下所示：

copytextpop-up
  1. select e.empno, e.ename, m.ename
  2. from emp e, emp m
  3. where e.mgr = m.empno;
select e.empno, e.ename, m.ename
from emp e, emp m
where e.mgr = m.empno;

2 练习使用外连接查询

2.1 问题

查询emp表，dept表及salgrade表的信息，完成下列SQL语句的编写：

1）查询dept表中所有的数据，及emp表中对应部门下员工的编号，姓名，职业；

2）查询emp表中所有的员工的信息及与之关联的dept表的部门名称，地址；

3）查询dept表中所有部门的编号及名称，与之对应部门下的员工的编号，名字，职位，奖金，工资；

4）查询emp表中所有的数据，及dept表中对应部门的数据，和dept表中其他的数据。

2.2 方案

外连接：

解决的是一个都不能少的问题，是对内连接的补充

具备三种形式

左外连接 left outer join on …

右外连接 right outer join on …

全外连接 full outer join on …

Oracle特有语法：

要想让左边表的记录一个都不能少，在对方添加(+)

标准SQL：

外连接-左外连接：左表 left outer join 右表 on 连接条件

外连接-右外连接：左表 right outer join 右表 on 连接条件

外连接-全外连接：左表 full outer join 右表 on 连接条件

2.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：查询dept表中所有的数据，及emp表中对应部门下员工的编号，姓名，职业

查询dept表中所有的数据，及emp表中对应部门下员工的编号，姓名，职业，SQL语句如下所示：

copytextpop-up
  1. select d.*,e.empno,e.ename,e.job
  2. from emp e,dept d
  3. where d.deptno = e.deptno (+); 
select d.*,e.empno,e.ename,e.job
from emp e,dept d
where d.deptno = e.deptno (+);

步骤二：查询emp表中所有的员工的信息及与之关联的dept表的部门名称，地址

查询emp表中所有的员工的信息及与之关联的dept表的部门名称，地址，SQL语句如下所示：

copytextpop-up
  1. select e.*, d.dname, d.loc
  2. from emp e left join dept d
  3. on e.deptno = d.deptno; 
select e.*, d.dname, d.loc
from emp e left join dept d
on e.deptno = d.deptno;

步骤三：查询dept表中所有部门的编号及名称，与之对应部门下的员工的编号，名字，职位，奖金，工资

查询dept表中所有部门的编号及名称，与之对应部门下的员工的编号，名字，职位，奖金，工资，SQL语句如下所示：

copytextpop-up
  1. select d.deptno, d.dname, e.empno, e.ename, e.job, e.comm,e.sal
  2. from emp e right outer join dept d
  3. on e.deptno = d.deptno; 
select d.deptno, d.dname, e.empno, e.ename, e.job, e.comm,e.sal
from emp e right outer join dept d
on e.deptno = d.deptno;

步骤四：查询emp表中所有的数据，及dept表中对应部门的数据，和dept表中其他的数据

查询emp表中所有的数据，及dept表中对应部门的数据，和dept表中其他的数据，SQL语句如下所示：

copytextpop-up
  1. select e.*,d.*
  2. from dept d full outer join emp e
  3. on e.deptno = d.deptno;
select e.*,d.*
from dept d full outer join emp e
on e.deptno = d.deptno;
