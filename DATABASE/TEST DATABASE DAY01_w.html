1 练习使用SQL Developer

1.1 问题

安装并练习使用SQL Developer操作Oracle数据库。

1.2 方案

我们可以使用SQL Plus来操作Oracle数据库，但是会发现使用起来及其的不方便，在实际的企业开发，一般使用SQL Developer来操作数据库。

SQL Developer是Oracle公司出品的免费图形化开发工具，相对SQL Plus来说，图形化的界面更易于操作，输出效果美观。

5 练习使用update语句

5.1 问题

练习使用update语句，修改emp表中的信息：

1、修改编号是6666的雇员的名字为李靖；

2、修改编号是9999的雇员sal为10000，奖金为30000。

5.2 方案

更新表中数据的语法：

update 表名set 列名1=该列新值，列名2=该列新值，…… where 条件；

如果不指定where条件，将把表中所有记录的要更新的列的值都更新成相同的新值

可以同时更新多列。

5.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：修改编号是6666的雇员的名字为李靖

修改编号是6666的雇员的名字为李靖，SQL语句如下所示：

copytextpop-up
  1. update emp set ename = '李靖'
  2. whereempno = 6666
update emp set ename = '李靖'
whereempno = 6666

步骤二：修改编号是9999的雇员sal为10000，奖金为30000

修改编号是9999的雇员sal为10000，SQL语句如下所示：

copytextpop-up

  2. whereempno = 9999
updateemp set sal = 10000,comm = 30000
whereempno = 9999

6 练习使用delete语句

6.1 问题

练习使用delete语句，删除emp表中及dept表中的数据：

1）删除emp表中编号是6666的雇员的信息；

2）删除emp表中编号是9999的雇员的信息；

3）删除dept表中所有的信息。

6.2 方案

使用delete删除：

delete from 表名where 条件；

说明：

1）如果不指定where条件，将把表中所有记录都删除掉。

2）如果使用where指定条件，则满足条件的记录被删除。

6.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：删除emp表中编号是6666的雇员的信息

删除emp表中编号是6666的雇员的信息，SQL语句如下所示：

copytextpop-up
  1. delete from emp where empno = 6666
delete from emp where empno = 6666

步骤二：删除emp表中编号是9999的雇员的信息

删除emp表中编号是9999的雇员的信息，SQL语句如下所示：

copytextpop-up
  1. delete from emp where empno = 9999
delete from emp where empno = 9999

步骤三：删除dept表中所有的信息

删除dept表中所有的信息，SQL语句如下所示：

copytextpop-up
  1. delete from dept
delete from dept

完整SQL语句

本案例中，完整SQL语句如下所示：

copytextpop-up
  1. --删除emp表中编号是6666的雇员的信息
  2. delete from emp where empno = 6666
  3. --删除emp表中编号是9999的雇员的信息

  5. --删除dept表中所有的信息
  6. delete from dept;
