1 方法重写的综合应用

1.1 问题

设计Person（父类）：

属性：name（姓名），sex（性别），age（年龄），address（地址）。

构造方法：

1)空参构造方法。

2）给所有的属性赋值的构造方法。

方法：show():显示所有的属性。

设计Emp（子类）:

属性：name（姓名），sex（性别），age（年龄）,address（地址）,salary（工资），comm（奖金）。

构造方法：给所有的属性赋值。

方法：show():显示所有的属性（重写）。

ShowYearSalary（）：salary*12+comm。

1.2 方案

设计Person类，首先编写属性：name（姓名），sex（性别），age（年龄），address（地址）。

然后编写构造方法，给Person中所有的属性赋值和空参的构造方法。之后再编写show方法，用于显示Person类中所有的属性值。

设计Emp类，继承Person类，首先编写属性：salary（工资），comm（奖金）。

然后编写构造方法，给Emp中所有的属性赋值。之后再重写show方法，最后编写。

最后在main方法中创建该Emp类的对象，然后给所有的属性赋值，然后调用show方法，显示所有的属性值及showYearSalary方法显示年薪。

2 super的简单应用

2.1 问题

设计Koo1类，方法：function(int num1,int num2),执行加法运算。

设计Koo2类，方法：function(int num1,int num2),执行乘法运算，function2(int num1,int num2)：分别调用Koo1中的function(int num1,int num2)方法和Koo2中function(int num1,int num2)方法。

2.2 方案

设计Koo1类，编写方法：function(int num1,int num2)，加法运算。

设计Koo2类继承Koo1，重写function(int num1,int num2)，乘法运算，编写function2(int num1,int num2)方法，利用super.function(num1，num2)调用父类Koo1中的function方法，再利用this.function(num1,num2)调用Koo2中的function方法。
