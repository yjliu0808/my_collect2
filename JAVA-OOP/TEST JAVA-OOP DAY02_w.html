方法：show方法，用于显示Person类中所有的属性值，showYearSalary方法显示年薪。

在main中创建该类的对象。

1.2 方案

设计Emp类，首先编写属性：名字（name），年龄（age），性别（sex），工资（salary），奖金（comm）。

3 使用构造方法重载重新设计Box类

3.1 问题

设计Box类，其中属性：长(length)，宽(width)，高（height）；

构造方法：1）带三个参数，分别给长宽高赋值；

2）带一个参数，给长宽高都赋同一个值；

方法：getVolume()，用于获得体积，在main中创建该类的对象。

3.2 方案

首先设计Box类，属性：length，widht，height，然后编写构造方法要注意，这次要用到构造方法重载，分别是带3个参数的构造方法和带1个参数的构造方法，来创建长方体对象和正方体对象，最后编写getVolume()方法，该方法要求返回的是体积的数值，求体积的公式：长*宽*高，把该体积的值返回到main方法中显示。

run():行驶，显示当前车名和速度。
