1 创建数组并访问数组中元素

1.1 问题

创建一个整型数组ary，有50个元素，把1~100之间所有的偶数给ary数组元素赋值并且倒序显示元素的值(去掉0)。

1.2 方案

首先创建整型数组，然后利用for给数组中的每一个元素赋值，然后再使用for循环倒序的输出数组的中的每一个元素。

4 编写一个带返回值的add方法

4.1 问题

设计一个方法add()，在add()方法中，定义两个变量num1，num2，从控制台输入两个数值给num1和num2分别赋值，然后执行加法运算，并把结果返回到main方法中显示。

4.2 方案

编写add的方法的时候，首先要注意返回值类型是int类型，然后在该方法内创建Scanner对象，从控制台输入两个整数的值，执行加法运算，并把最终的结果返回到调用的问题，显示出来。

4.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：定义类及add方法

首先定义一个名为Demo4的类，声明add方法，代码如下所示：

copytextpop-up

  2.     static int add（）{        
  3.     } 
  4. }
public class Demo4{
	static int add（）{		
	} 
}

步骤二：创建Scanner对象

根据引用类型来创建该引用类型的对象。

Scannerscanner = new Scanner(System.in)，代码如下所示：
