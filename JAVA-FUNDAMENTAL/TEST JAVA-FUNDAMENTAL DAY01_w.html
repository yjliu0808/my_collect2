1 控制台版的JAVA HelloWorld

1.1 问题

StringBuffer的常用方法，toString方法和append方法的使用。

使用记事本编写HelloWorld.java程序，在控制台输出“Hello World”。运行界面如图-1所示：

图-1

1.2 方案

1．在D:\盘下新建一个HelloWorld.java文本文件

  • HelloWorld是文件名，需要注意大小写
  • .java是后缀，或者称为扩展名
  • Windows系统通过后缀来区分不同的文件

2．编辑源程序

3．编译javac Java源文件的名称

  • javac HelloWorld.java
  • 在控制台中进入D:\盘，输入javac HelloWorld.java

4．运行控制台：java 类的名称

  • java HelloWorld

1.3 步骤

步骤一：

在D：\盘下创建HelloWorld.java文件，如下图-2所示:

图-2

步骤二：

使用纯文本编辑器记事本直接打开，编辑如下代码：

copytextpop-up
  1. public class HelloWorld {
  2.     public static void main(String[] args) {
  3.         System.out.println("Hello World");
  4.     }
  5. }
public class HelloWorld {
	public static void main(String[] args) {
		System.out.println("Hello World");
	}
}

在记事本中如下图-3所示：

图-3

步骤三：

由于源文件在D:\目录下，所以在控制台输入d: 让盘符显示为D:\

接着使用javac命令编译，javac空格后添加源文件的名称，如下图-4所示：

图-4

注意：如果编译通过，则不会提示错误信息，并且会在相应目录下生成HelloWorld.class字节码文件，表示一个Java类。

如果编译有错误， String写成string，如下图-5所示：

图-5

则在编译时会提示以下信息，如下图-6所示：

图-6

如上图-6所示，提示出HelloWorld.java源文件中的第2行，出现了编译错误，错误的原因是找不到符号。string类的名称找不到，而且给出了具体出错的位置。此时需要重新打开源文件进行编辑，直到更正后编译通过为止。

步骤四：

运行，在控制台中执行java命令，后添加类名HelloWorld，如下图-7所示：

图-7

运行成功后，会在控制台出现Hello World！内容。

  4.     }
  5. }

	public static void main(String[] args) {
		System.out.println("Hello World");
	}

2 使用Eclipse开发Java应用程序

2.1 问题

使用Eclipse编写Demo1.java程序，输出“努力学习，java”。

2.2 方案

首先创建一个工程Project，名称day01，然后在src下创建一个包com.tarena.demo,在该包下创建Demo1.java文件，最后Demo1中编写程序并执行。

2.3 步骤

步骤一：

首先指定工作区的目录：以后开发的项目代码的位置D:\Workspaces;

先创建一个项目Project，名称：javase;

创建方式：File -> new -> Java Project–>Project name: javase，如下图-8所示：

图-8

在Project name中填写项目的名称javase,其它设置选择默认，单击”Finish”即可创建好新的工程，如下图-9所示：

图-9

弹出以下窗口，是选择MyEclipse视图的含义，我们可以单击”No”选择默认的视图即可，如下图-10所示：

图-10

工程创建好后，可见左边Package Explorer窗口中体现出项目的名称和资源，其中主要的目录是src目录，用于创建新的Java包和源程序，如下图-11所示：

图-11

可以打开工作区目录查看，如下图-12所示：

图-12

在新建立的Java项目中有src和bin两个子目录：

  • javase
  • src 代表源代码的存放目录
  • bin 编译后.class文件的存放目录

3 定义变量初始化，并输出变量值

3.1 问题

定义三个变量num1，num2，num3，分别赋值10,20,30，并在控制台输出。

3.2 方案

首先创建main方法，然后在main方法中定义3个num变量，之后在定义的时候初始化，最后在控制台显示变量的名字和变量的值。

需要使用到变量的初始化，以及输出语句的使用。

3.3 步骤

步骤一：定义类及 main方法

首先定义一个名为Demo2的类，并在类中定义Java应用程序的入口方法main，代码如下所示：

copytextpop-up
  1. public class Demo2{
  2.     public static void main(String[] args){        
  3.     } 
  4. }
public class Demo2{
	public static void main(String[] args){		
	} 
}

步骤二：定义变量

定义三个变量num1，num2，num3，分别赋值10,20,30,代码如下所示：

copytextpop-up
  1. public class Demo2{
  2.     public static void main(String[] args){
  3.         int num1=10,num2=20,num3=30;
  4.     }
  5. }
public class Demo2{
	public static void main(String[] args){
		int num1=10,num2=20,num3=30;
	}
}

步骤三：输出变量值

在控制台输出变量名和对应变量的值，代码如下所示：

copytextpop-up
  1. public class Demo2{
  2.     public static void main(String[] args){
  3.         int num1=10,num2=20,num3=30;
  4.         System.out.println("num1="+num1);
  5.         System.out.println("num2="+num2);
  6.         System.out.println("num3="+num3);        
  7.     }
  8. }
public class Demo2{
	public static void main(String[] args){
		int num1=10,num2=20,num3=30;
		System.out.println("num1="+num1);
		System.out.println("num2="+num2);
		System.out.println("num3="+num3);		
	}
}
