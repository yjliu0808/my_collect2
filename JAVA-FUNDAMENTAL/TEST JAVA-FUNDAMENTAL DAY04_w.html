4 求出1+2+3...+100 的和

第1次：sum = sum + 1；

第2次：sum = sum + 2；

第3次：sum = sum + 3；

...

第100次：sum =sum + 100；

定义变量i，i的初值1，i的取值范围1~100；

归纳：sum = sum + i；

通过for来实现功能，i控制循环次数。

5 九九乘法表

5.1 问题

使用循环嵌套，实现打印九九乘法表。

1*1 = 1

1*2 = 2 2*2 = 4

1*3 = 3 2*3 = 6 3*3 = 9

…

5.2 方案

首先控制外层循环的次数（也就是打印的行数），然后再考虑内层循环的次数（每行表达式的个数），这个时候能发现一个规律，就是每行表达式的个数跟当前行数一致。

可以这样的编写循环语句，代码如下所示：

copytextpop-up
  1. for(int i=1;i<=9;i++){
  2.     for(int j=1;j<=i;j++){
  3.     }
  4. }
for(int i=1;i<=9;i++){
	for(int j=1;j<=i;j++){
	}
}
