1 使用整数类型时需要注意的问题

1.1 问题

实际的企业开发中，对于整数类型，我们最常用的是int和long，使用的时候需要注意，当给long类型的变量赋值的时候，如果字面量超过int的取值范围的时候，该如何进行处理。

1.2 方案

首先需要清楚，整型的字面量默认就是int类型，如果给long类型的变量赋值的时候，如果该字面量没有超过int的取值范围，则不用进行任何处理，如果一旦超过int的取值范围，则必须在末尾加上L或者l。

1.3 步骤

实现此案例需要按照如下步骤进行。

首先定义一个名为Demo1的类，并在类中定义Java应用程序的入口方法main，代码如下所示：

2 使用浮点类型时需要注意的问题

2.1 问题

2.2 方案

首先需要注意，浮点类型的字面量默认就是double类型，这样如果给double类型的变量赋值没有任何问题，但如果给float类型的变量赋值的时候，必须要在字面量的末尾加上f或者F。
