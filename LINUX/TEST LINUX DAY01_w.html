1 使用图形界面形式安装Linux

1.1 问题

熟悉Linux系统的安装，练习使用图形界面方式安装Linux系统。

1.2 方案

在虚拟机中使用图形界面方式安装Linux系统。

1.3 步骤

步骤一：加载安装文件

1、选择虚拟机名称，点击VM菜单，选择Settings选项，如下图-1所示：

图-1

2、在Virtual Machine Settings窗口中选择CD/DVD(IDE)选项，选择Use ISO image file:文件类型，点击Browser选择第一个安装文件，如下图-2所示：

图-2

3、点击OK 按钮，如下图-3所示：

图-3

4、点击Power on this virtual machin选项，启动虚拟机，如下图-4所示：

图-4

5、在显示的界面中点击回车按钮，如下图-5所示：

图-5
