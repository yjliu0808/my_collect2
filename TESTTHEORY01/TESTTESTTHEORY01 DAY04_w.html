2 使用判定表法分析企业工资发放软件

工资为YX的员工犯普通错误的扣工资4％（K3），犯严重错误扣工资8％（K4）。

员工编号：

1——1000的整数，不能为空。

姓名：

1——20个字符，不能为空。

本月工资：

0.00——100000.00的小数，小数点后最多保留2位，不能为空。

实现此案例需要按照如下步骤进行。

步骤一：找出所有的输入条件

3 使用判定表法分析字符组合程序

3.1 问题

有一个字符组合程序，需求如下：

输入的第一个字符必须为# 或 * ，第二个字符必须是一个数字，在此情况下进行文件修改；如果第一个字符不是#或*，则给出信息N；如果第二个字符不是数字，则给出信息M

N和M可以同时输出

暂时先不考虑其他情况，如第一个或第二个字符不填写情况

3.2 方案

1、找出所有的输入条件。

2、明确所有的输出结果。

3、明确所有条件之间关系。

4、确定因果关系，写出判定表。

5、编写测试用例。

3.3 步骤

实现此案例需要按照如下步骤进行。

步骤一：找出所有的输入条件

找出企字符组合程序的所有输入条件：

（1）第一个字符为#

（2）第一个字符为*

（3）第一个字符为其他（#*以外的）

（4）第二个字符为数字

（5）第二个字符为非数字

步骤二：明确所有的输出结果

A、修改文件

B、给出信息N

C、给出信息M

步骤三：明确所有条件之间的关系

（1）限制：

输入（1）（2）（3）不能组合

输入（4）（5）不能组合

（2）组合 ：

输入（1）（4）组合

输入（1）（5）组合

输入（2）（4）组合

输入（2）（5）组合

输入（3）（4）组合

输入（3）（5）组合

步骤四：确定因果关系，写出判定表

找到输入组合和输出组合的对应关系

输入（1）（4）组合 => 输出A

输入（1）（5）组合 => 输出C

输入（2）（4）组合 => 输出A

输入（2）（5）组合 => 输出C

输入（3）（4）组合 => 输出B

输入（3）（5）组合 => 输出BC
